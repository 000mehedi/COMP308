        #Prompt: Describe the 4 levels of access specification in Java
            public: public allows to make accessible to anywhere
            private: private restricts the accessibility to anyone
            protected: protected allows child classes to access protected members of the parent class
            package: package keyword allows dividing classes themselves into packages, between which they cannot access each other.

        #Prompt: Purpose of iterator
            An Iterator is an object that can be used to loop through collections, like ArrayList and HashSet.
            It is called an "iterator" because "iterating" is the technical term for looping.
